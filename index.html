01

Basic Input & Output

OUTPUT

Printing to Screen

System.out.println() prints and moves to a new line. System.out.print() stays on the same line.

INPUT

Reading from Keyboard

Import java.util.Scanner, create a Scanner object, then use methods like nextLine(), nextInt(), nextDouble().

JAVA — Names Input/Output 
import java.util.Scanner;

public class NameExample {
    public static void main(String[] args) {

        Scanner keyboard = new Scanner(System.in);

        // Ask for and read a name
        System.out.print("Enter your first name: ");
        String firstName = keyboard.nextLine();

        System.out.print("Enter your last name: ");
        String lastName = keyboard.nextLine();

        // Print the result
        System.out.println("Hello, " + firstName + " " + lastName + "!");

        // String concatenation with a number
        int age = 20;
        System.out.println(firstName + " is " + age + " years old.");

        keyboard.close();
    }
}
⚡ TIP: Use nextLine() for text (String). Use nextInt() for whole numbers. Use nextDouble() for decimals. Never forget to close the scanner!
JAVA — Reading Multiple Inputs 
// Storing multiple names in an array
System.out.print("How many students? ");
int count = keyboard.nextInt();
keyboard.nextLine();  // clear leftover newline after nextInt()

String[] names = new String[count];

for (int i = 0; i < count; i++) {
    System.out.print("Enter name " + (i + 1) + ": ");
    names[i] = keyboard.nextLine();
}

// Print all names
for (String name : names) {
    System.out.println(name);
}

04

Data Type Conversions

WIDENING (Implicit)

Automatic Conversion

Java automatically converts a smaller type to a larger type. No data loss. No casting needed.

byte → short → int → long → float → double

NARROWING (Explicit)

Manual Casting Required

Converting a larger type to a smaller type. You must cast manually. May lose precision or data!

double → float → long → int → short → byte

JAVA — Type Casting Examples 
// ── WIDENING (automatic) ──────────────────
int    i = 100;
long   l = i;        // int  → long  ✓ automatic
double d = i;        // int  → double ✓ automatic
System.out.println(d); // Output: 100.0

// ── NARROWING (must cast manually) ────────
double pi   = 3.9999;
int    piInt = (int) pi;   // truncates — does NOT round!
System.out.println(piInt); // Output: 3  (not 4)

double score = 95.7;
int    rounded = (int) Math.round(score); // use Math.round to round
System.out.println(rounded); // Output: 96

// ── String ↔ Number conversions ──────────
String numStr = "42";
int    num    = Integer.parseInt(numStr);    // String → int
double dbl    = Double.parseDouble("3.14"); // String → double
String back   = String.valueOf(num);          // int → String
String back2  = "" + num;                   // shortcut: int → String

// ── char ↔ int ───────────────────────────
char c = 'A';
int  ascii = (int) c;    // char → int: 65
char back3 = (char) 66;  // int → char: 'B'
⚡ TIP: Casting with (int) always truncates (chops off) decimals. Use Math.round() when you actually need rounding!

✅ FIXED 
import java.util.Scanner;  // 1: missing ;

public class Fix {
  public static void main(String[] args) {
    Scanner sc = new Scanner(System.in);  // 2: missing ;

    System.out.print("Enter marks: ");
    int marks = sc.nextInt();  // 3: nextint → nextInt

    int[] results = new int[marks];  // 4: missing 'new'
    for (int i = 0; i < marks; i++) {  // 5: commas → semicolons
      results[i] = sc.nextInt();
    }

    for (int j = 0; j < results.length; j++) {
      System.out.println(results[j]);  // 6: Results → results
    }
  }
}
⚡ Common mistakes checklist: 1) Missing semicolons 2) Wrong capitalisation (Java is case-sensitive) 3) Commas instead of semicolons in for-loops 4) Missing new when creating arrays 5) Wrong Scanner method names

Pseudocode ↔ Java

Example 1 — Calculate average of N numbers

PSEUDOCODE 
BEGIN
  INPUT n
  SET total = 0
  FOR i = 1 TO n
    INPUT num
    SET total = total + num
  END FOR
  SET average = total / n
  OUTPUT average
END
JAVA 
Scanner sc = new Scanner(System.in);

System.out.print("Enter n: ");
int n = sc.nextInt();
double total = 0;

for (int i = 1; i <= n; i++) {
    System.out.print("Number: ");
    total += sc.nextDouble();
}

double average = total / n;
System.out.println("Average: " + average);

Example 2 — Check if number is even or odd

PSEUDOCODE 
BEGIN
  INPUT number
  IF number MOD 2 = 0 THEN
    OUTPUT "Even"
  ELSE
    OUTPUT "Odd"
  END IF
END
JAVA 
Scanner sc = new Scanner(System.in);

System.out.print("Enter number: ");
int number = sc.nextInt();

if (number % 2 == 0) {
    System.out.println("Even");
} else {
    System.out.println("Odd");
}
